# 🎓 Memory Lane

A digital platform to capture, relive, and preserve BTech memories — from first year to final farewell.

---

## ✨ Overview

Memory Lane is a modern web platform designed to document the journey of a college batch through:

- 📸 Photos & videos  
- 👥 Student yearbook  
- 🕰️ Semester timeline  
- 💬 Message wall  

It’s not just a gallery — it’s a **digital time capsule**.

---

## 🚀 Features

### 🏠 Landing Experience
- Cinematic hero section
- Emotional storytelling UI

### 🕰️ The Journey
- Semester-wise timeline
- Organized memory flow

### 👥 Yearbook
- Student directory
- Search & hover interactions
- Instagram tagging

### 📸 Media Vault
- Event-based gallery
- Photos + videos
- Smart categorization

### 💬 The Wall
- Message board (like Pinterest)
- Final notes from classmates

---

## 🛠 Tech Stack

- ⚛️ Next.js  
- 🎨 Tailwind CSS  
- 🗄️ Supabase (Auth + DB + Storage)  
- 🎬 Framer Motion  
- ☁️ Vercel (Hosting)

## ⚙️ Setup Instructions

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/memory-lane.git

# Go into project
cd memory-lane

# Install dependencies
npm install

# Run the app
npm run dev

🔐 Environment Variables

Create a .env.local file:

NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
